Knitting and crocheting are well loved hobbies that many people of all ages and genders enjoy. Since these hobbies are handcrafts, creating projects can take a long time. As a result, the novelty of a project often wears off before it’s finished, leading to a decrease in motivation to continue the project. For example, when knitting a pair of socks, one may finish one sock and not have the motivation to finish the second one, which is a common occurrence that has been dubbed “second sock syndrome” (Lonesome George & the Second Sock Syndrome | Schachenmayr, n.d.). Many people end up putting down the project and starting another one, thus finding themselves with a forgotten pile of works-in-progress (WIPs) that can cause anxiety, similar to how a long to-do list does.

There are many different websites and apps that aid in knitting and crocheting. Ravelry is a widely used website that has many different features. A specific feature that Ravelry has that relates to the issue described is a project tracker that allows the user to add projects and input information about them, such as the materials used and hook size. Despite its popularity, there is no official mobile app for Ravelry (Ravelry: Apps That Connect to Ravelry, n.d.). The issue that I will be addressing with this project is the difficulty in maintaining motivation and keeping track of multiple projects simultaneously. In this project, I will create a mobile app that improves upon Ravelry’s project tracker by including a motivational aspect (achievements). The goal of this project is to create a convenient application that helps maintain motivation throughout projects.
